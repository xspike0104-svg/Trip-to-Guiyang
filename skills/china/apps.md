# Useful Apps for China Travel

## Transport and Navigation

- **Rail and flight official apps** for schedule changes
- **Major map apps** with offline backup layers
- **Ride-hailing apps** for late-night returns and station transfers

## Booking and Dining

- **Restaurant booking platforms** for high-demand cities
- **Hotel and transport operator apps** for disruption alerts

## Translation and Practical

- **Translation and camera text tools** for menus and signs
- **Payment-wallet linked apps** for daily spending operations

## Workflow Tip

Install and verify core apps before departure, then keep offline backups for maps and key reservation data.
